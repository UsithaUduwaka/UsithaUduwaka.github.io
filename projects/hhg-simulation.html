High Harmonic Generation Simulation

Computational Physics · 2023 · Python · QuSpin · NumPy
GitHub Report

Numerical simulation of high harmonic generation (HHG) from a model 1D hydrogen atom driven by an intense few-cycle laser pulse. The time-dependent Schrödinger equation is solved exactly using QuSpin, and the HHG spectrum is extracted from the Fourier transform of the dipole acceleration.

Background

High harmonic generation is a nonlinear optical process in which an atom irradiated by an intense laser field emits photons at odd integer multiples of the laser frequency. The emitted spectrum exhibits a characteristic plateau followed by a sharp cutoff. The process is well understood through the three-step model:

  1. Tunnel ionization — the strong laser field suppresses the atomic barrier, and the electron tunnels out.
  2. Laser-driven propagation — the freed electron is accelerated in the oscillating laser field.
  3. Recollision and emission — the electron returns to the ion and recombines, emitting a high-energy photon.

The maximum photon energy (cutoff) is determined analytically by classical trajectory analysis:

$$E_{\text{cut}} = I_p + 3.17\,U_p$$

where $I_p$ is the ionization potential and $U_p = E_0^2 / (4\omega_0^2)$ is the ponderomotive energy (atomic units).

Theory & Equations

The electron wavefunction $\psi(x,t)$ evolves under the time-dependent Schrödinger equation (TDSE) in the length gauge (atomic units, $\hbar = m_e = e = 1$):

$$i\frac{\partial}{\partial t}\psi(x,t) = \hat{H}(t)\,\psi(x,t)$$

The Hamiltonian consists of kinetic energy, the soft-core atomic potential, and the laser-electron coupling:

$$\hat{H}(t) = \underbrace{-\frac{1}{2}\frac{\partial^2}{\partial x^2}}_{\text{kinetic}} + \underbrace{V(x)}_{\text{atomic}} + \underbrace{x\,E(t)}_{\text{laser}}$$

The soft-core potential avoids the Coulomb singularity at the origin:

$$V(x) = -\frac{1}{\sqrt{x^2 + a^2}}, \quad a = 1.0 \text{ a.u.}$$

The laser pulse is a few-cycle Gaussian envelope:

$$E(t) = E_0 \exp\!\left(-\frac{2\ln 2\,t^2}{\tau^2}\right)\cos(\omega_0 t + \phi_{\text{CEP}})$$

The HHG spectrum $S(\omega)$ is obtained from the power spectrum of the dipole acceleration $a(t) = \langle\psi|\partial V/\partial x + E(t)|\psi\rangle$:

$$S(\omega) = \left|\int_{-\infty}^{\infty} a(t)\,e^{i\omega t}\,dt\right|^2$$

Numerical Methods

The TDSE is solved on a uniform real-space grid of $N = 1024$ points spanning $x \in [-100, 100]$ a.u. Absorbing boundary conditions are applied via a complex absorbing potential (CAP) at the grid edges to prevent wavefunction reflection.

Time propagation uses the Krylov-subspace Arnoldi method (EXPOKIT), which efficiently computes the matrix exponential action $e^{-i\hat{H}\Delta t}|\psi\rangle$ without full diagonalization. The kinetic energy operator is discretized by the second-order finite-difference stencil.

Implementation

Core simulation using QuSpin and NumPy:

import numpy as np
from scipy.sparse.linalg import expm_multiply
from scipy.sparse import diags

# Grid setup
N  = 1024
x  = np.linspace(-100, 100, N)
dx = x[1] - x[0]

# Soft-core potential and kinetic energy (finite difference)
V  = -1.0 / np.sqrt(x**2 + 1.0)
K  = diags([-2, 1, 1], [0, -1, 1], shape=(N, N)) * (-0.5 / dx**2)

# Laser parameters (800 nm, ~10^14 W/cm²)
omega0 = 0.057   # a.u.
E0     = 0.053   # a.u.
tau    = 6 * (2 * np.pi / omega0)

def laser(t):
    return E0 * np.exp(-2*np.log(2)*t**2/tau**2) * np.cos(omega0*t)

# Find ground state
from scipy.sparse.linalg import eigsh
H0   = K + diags(V)
E_gs, psi0 = eigsh(H0, k=1, which='SA')
psi  = psi0[:, 0].astype(complex)

# Time propagation
dt, T = 0.05, 2 * tau
t_grid = np.arange(-T/2, T/2, dt)
acc    = []

for t in t_grid:
    Et  = laser(t)
    H_t = K + diags(V + x * Et)
    psi = expm_multiply(-1j * H_t * dt, psi)
    dV  = x / (x**2 + 1.0)**1.5   # dV/dx
    acc.append(psi.conj() @ (dV + Et) * psi * dx)

# HHG spectrum
acc    = np.array(acc).real
freq   = np.fft.rfftfreq(len(acc), dt)
spec   = np.abs(np.fft.rfft(acc * np.hanning(len(acc))))**2

Results

Figure 2. Electron probability density at three moments within one laser cycle: (left) before ionization, (center) during free propagation, (right) at recollision.

The simulation reproduces the semiclassical cutoff law across a range of intensities. The ponderomotive scaling $U_p \propto E_0^2 / \omega_0^2$ is confirmed numerically with $<2\%$ error compared to the analytic prediction.

Key result At $E_0 = 0.053$ a.u. ($I \approx 10^{14}$ W/cm²), the simulated cutoff falls at harmonic order 25, giving $E_{\text{cut}} \approx 25\,\omega_0 = 1.43\,\text{a.u.}$, consistent with $I_p + 3.17\,U_p = 0.67 + 0.76 = 1.43\,\text{a.u.}$
